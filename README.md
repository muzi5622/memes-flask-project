A simple Flask web app that fetches random memes from popular Reddit subreddits and displays them on a webpage.

---

## Features
- Fetches memes from multiple subreddits (e.g., `memes`, `dankmemes`, `ProgrammerHumor`).
- Skips NSFW content automatically.
- Refresh the page to get a new random meme.
- Lightweight and easy to run.

---

## Requirements
- Python 3.x
- Flask
- requests



## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/muzi5622/memes-flask-project.git
   cd memes-flask-project


2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

---

## Usage

1. Run the Flask app:

   ```bash
   python app.py
   ```

2. Open your browser and go to:

   ```
   http://127.0.0.1:5000
   ```

3. Enjoy your memes ðŸŽ‰
